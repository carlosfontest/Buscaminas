# Buscaminas (Juego) [JAVA]

## ***Introducción***
Juego clásico de *Buscaminas*, con las mismas funcionalidades que tiene el juego original de Windows. El objetivo del juego es *despejar* un campo de minas *sin detonar* ninguna.
